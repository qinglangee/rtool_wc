wc 工具，可以数中文的字数

一个用来数字数和行数的工具，类似 wc 的功能
- 如果没有参数运行，就打印帮助信息。
- 如果参数是文件就读取文件内容，计算行数和字数和字节数，类似 wc 输出结果和文件名， 文件名用 `[]` 包围。
- 英文字符、西方文字和数字同 wc 工具类似， 以英文单词计数方式计数单词。
- CJK 字符以单字计数， 每个字符计算为一个单词。
- 有一个配置文件可以读取忽略字符配置， 当参数要求忽略方式计数时，忽略字符不计算个数。
- 配置文件不存在时，默认没有忽略字符
- 配置参数有 -i 时为忽略方式计数。
- 配置参数有 -z 时为中文计数， 只计算中文字符的个数。
